TOOL BMS

ESP32 Web Flasher

Flash firmware directly from this page.

Use Chrome or Microsoft Edge on Windows/macOS/Linux. Connect the ESP32 by USB, then press Install.

This browser does not support Web Serial. Use Chrome or Edge. Web Serial is blocked. Open this page with HTTPS or localhost.

First install can erase ESP32 flash.

After flashing, the device will reboot. If the serial port is not visible, install the CP210x or CH340 USB driver and reconnect the board.

Firmware files are loaded from manifest.json in this GitHub Pages folder.
